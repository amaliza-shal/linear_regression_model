# Student Exam Score Predictor

## Mission
To predict student exam scores based on academic and personal factors using machine learning, helping educators identify at-risk students early and provide targeted support.

## Problem Statement
Predicting student exam performance based on multiple factors including access to resources, teacher quality, parental involvement, and socioeconomic background using machine learning.

## Dataset
- **Source**: Kaggle - StudentPerformanceFactors.csv
- **Link**: https://www.kaggle.com/datasets/lainguyn123/student-performance-factors
- **Size**: 6,607 students with 20 features
- **Features**: Hours studied, attendance, parental involvement, access to resources, teacher quality, school type, family income, motivation level, and more

## Models Used
- Linear Regression (Final Production Model)
- Decision Tree Regressor
- Random Forest Regressor

## Public API
The trained model is deployed on Render and available at:
```
https://student-exam-predictor-v1.onrender.com
```

## Flutter Mobile App
A Flutter app is included for easy mobile access to the prediction API.

### How to Run the Flutter App
1. Install Flutter from https://flutter.dev
2. Clone this repository:
   ```bash
   git clone https://github.com/amaliza-shal/linear_regression_model.git
   cd linear_regression_model
   ```
3. Navigate to the Flutter app folder:
   ```bash
   cd summative/flutter_app
   ```
4. Get dependencies:
   ```bash
   flutter pub get
   ```
5. Run the app:
   ```bash
   flutter run
   ```
   Choose your Android emulator or connected device when prompted.
