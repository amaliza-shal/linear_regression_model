Mission
My mission is mainly to empower children from underserved communities by providing 
access to quality education through government support, qualified 
teachers and mentorship to help them succeed in life.

Problem
Predicting student exam performance based on access to resources,
teacher quality and socioeconomic background using machine learning.

Dataset
- Source: Kaggle
- Link: https://www.kaggle.com/datasets/lainguyn123/student-performance-factors
- Contains 6608 students with 20 features including teacher quality,
  parental involvement, access to resources and exam scores.

Models Used
- Linear Regression
- Decision Tree
- Random Forest.
